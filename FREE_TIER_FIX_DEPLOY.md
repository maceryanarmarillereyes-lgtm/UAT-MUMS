## ✅ What Was Fixed (Summary)

| Issue | Root Cause | Fix Applied |
|-------|-----------|-------------|
| 2,072 Auth requests/hour | `getUserFromJwt()` called on EVERY heartbeat | Server-side JWT cache (4min TTL) in `server/lib/supabase.js` |
| Disk IO Budget depleting | `mums_presence` LOGGED → 57,600 WAL writes/day | `SET UNLOGGED` via SQL (see below) |
| DB/PostgREST Unhealthy | REPLICA IDENTITY FULL on subscribed tables | `REPLICA IDENTITY DEFAULT` via SQL |
| Heartbeat index bloat | Unbounded INSERT growth | Single-row UPSERT + trim trigger |
| Mailbox polling too fast | `MAILBOX_OVERRIDE_POLL_MS=10000` (10s) | Changed to `60000` (60s) → idle=360s |
| Profile reads per list call | `getProfileForUserId()` called per user per poll | Profile cache (5min TTL) in `server/lib/supabase.js` |

```sql
-- relpersistence should be 'u' (unlogged), not 'p' (permanent)
```


⚡ How the Hero Counter works:
Primary (automatic): Uses the same Quickbase monitoring feed as My Quickbase — automatically filtered to each user's QB Name. Counts are always identical to what each user sees in My Quickbase. No field configuration required.
Fallback: If the monitoring adapter is unavailable, the system uses the Assigned-To Field above (advanced config). Open the "Advanced" section above only if needed.

Global Mailbox Switch
Mailbox Feature
When disabled: the Mailbox page shows a "temporarily disabled" notice for all users. All Supabase and Cloudflare requests from the mailbox (sync, assignment, real-time timers) are completely stopped — protecting your free-tier quota.
⚠️ Disabling the Mailbox will:
  • Stop all roster sync, assignment, and real-time timer requests
  • Stop periodic schedule re-sync (every 90s) to Supabase
  • Stop presence updates triggered by the mailbox
  • Show "Mailbox has been temporarily disabled" to all users
  • Super Admin can re-enable at any time from this panel
